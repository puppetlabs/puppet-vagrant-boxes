Overview

These boxes are provided to be used by various Puppet projects, so we have a consistent source for testing and other Vagrant work.

We use Veewee to build all of our systems, gladly contributing to the upstream project where necessary. If you haven't tried out this awesome tool yet, we highly recommend it.

If you want to raise bugs or work on the Veewee sources for these boxes, go to the puppetlabs/puppet-vagrant-boxes on Github.

These hosts are also indexed from: http://vagrantbox.es/ a place where you can find other vagrant images if you can't find what you need here.
